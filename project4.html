Expressive Text Art
This project was part of my Interaction Design class during Fall 2021 semester. The prompt was to make an interactive website, through which can scroll for some music lyrics, poem, texts, etc.
Ideation
The idea was to scroll the poem "Mirror" by Sylvia Plath with some imagery of mirror. The idea was to create the contrast between the mirror and water, something which the poet highlights. I also wanted to take this opportunity to experiment with CSS and create something experimental and artsy.
Execution
I am using HTML, CSS, and JavaScript to execute this project. The initial title fades away on scroll to emphasize the fading away when we stare into water. The poem text uses image mask effect on set of three mirrors to ironically create the water ripple effect whilst we scroll down to read the poem.
Lastly, I am using brackets to resemble the mirror and somewhat expose them through image mask as a suprise or the big reveal of irony. The code of this project is publicly available on GitHub and the project is hosted on the internet using GitHub Pages.
