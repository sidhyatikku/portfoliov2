Student Interview Project
For this project, we were paired with a classmate and asked to fill out 10 questions and answers. After we completed 10 questions and answers, we had to make an "interview" website from scratch for each other.
Ideation
The initial idea was create a simple and easy-to-use website with "card-like" interface from which user can navigate between questions and answers. The user can move between questions like a presentation and exit to home page at any time. I also wanted the website to look modern and have clean UI.
Execution
I made this website solely using HTML and CSS. Apart from home page, the questions have the same code, just different content inside divs. The website is somewhat responsive since I'm not using absolute units like pixels. The code is publicly available on GitHub and the website is hosted using GitHub Pages.
