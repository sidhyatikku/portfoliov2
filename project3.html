About Me Webpage Design
This project was made for my Interaction Design class during Fall 2021 semester. We had to design an "about me" page on Figma and replicate that using HTML and CSS.
Ideation
The initial idea behind the design was to create a basic about me page which I can put on my portfolio website. I also wanted to take this oppurtunity to test my CSS skills as I'm working flex-box, gradient, etc.
I made the initial UI design on Figma and simply coded that design. The design evolved as I was coding the webpage to the final product below.
The code of this project is publicly available on GitHub and the project is hosted on the internet using GitHub Pages.
